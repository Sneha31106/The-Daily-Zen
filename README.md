# The Daily Zen (Inspiration)

## Description
A minimal mental health dashboard designed to help you focus and find peace. It provides a beautiful daily quote, a serene background image, and a single "Focus Task" for the day.

## Core Features
*   **Random Quote Generator:** Get inspired by daily quotes.
*   **Serene Backgrounds:** Integration with Unsplash for calming, high-quality background images.
*   **Focus Task:** A simple, one-item to-do list to keep you focused on what matters most.
*   **Zen Interface:** Ultra-minimalist UI with smooth fading transitions for a relaxing experience.
*   **Breathing Circle:** A dedicated 1-minute expanding and contracting animation to guide mindful breathing.

## Tech Stack & Concepts
*   **Frontend Framework:** React
*   **Key React Concepts:**
    *   Combining multiple API responses into a single, cohesive UI view.
    *   Using `useEffect` with empty dependency arrays for initial data fetching and component mounting.

## APIs Used
*   [Unsplash API](https://unsplash.com/developers) - For fetching beautiful background imagery.
*   [ZenQuotes API](https://docs.zenquotes.io/zenquotes-documentation/) - For fetching random daily inspirational quotes.
